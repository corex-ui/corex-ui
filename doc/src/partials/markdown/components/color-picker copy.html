# Color Picker

A pure HTML and vanilla JS implementation of Zag JS Color Picker

An color-picker is a vertically stacked set of interactive headings containing a title, content snippet, or thumbnail representing a section of content.

# Anatomy

The Color Picker component consists of the following data parts:

root, item, item-content, item-trigger, item-indicator
