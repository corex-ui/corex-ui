# Effect Tokens

Design tokens for visual effects including shadows, blur, easing curves, and animations. These tokens provide foundational visual feedback across UI components using Tailwind v4 theme variables.

# Shadow

Use utilities like shadow-sm and shadow-lg to apply different sized outer box shadows to an element: 

<button class="shadow-2xs button">2xs shadow</button>
<button class="shadow-xs button">xs shadow</button>
<button class="shadow-sm button">sm shadow</button>
<button class="shadow-md button">md shadow</button>
<button class="shadow-lg button">lg shadow</button>
<button class="shadow-xl button">xl shadow</button>
<button class="shadow-2xl button">2xl shadow</button>

# Inset Shadow

Use utilities like inset-shadow-xs and inset-shadow-sm to apply an inset box shadow to an element: 

<button class="inset-shadow-2xs button">2xs inset</button>
<button class="inset-shadow-xs button">xs inset</button>
<button class="inset-shadow-sm button">sm inset</button>

<button class="drop-shadow-md button">md drop shadow</button>

# Easing Curves

Use utilities like ease-in and ease-out to control transition timing functions:

<button class="transition-transform hover:scale-105 ease-out button">ease out</button>
<button class="transition-transform hover:scale-105 ease-in-out button">ease in-out</button>
